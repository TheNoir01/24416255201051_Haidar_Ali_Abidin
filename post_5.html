Tutorial Install phpmyadmin via laragon

Posted by Haidar Ali Abidin on October 10, 2025

Halo, para pejuang kode!

Sering merasa sedikit ribet saat harus setup local server untuk proyek web baru? Harus install Apache, PHP, MySQL satu per satu, lalu konfigurasinya kadang bikin pusing. Nah, di artikel kali ini, saya mau bagikan cara super mudah untuk menyiapkan development environment yang powerful hanya dalam beberapa menit menggunakan Laragon.

Kita akan menggunakan Laragon versi 6, yang merupakan versi terakhir yang sepenuhnya gratis, dan tentu saja kita akan lengkapi dengan phpMyAdmin untuk mengelola database kita dengan mudah.

Install Laragon

Langkah 1: Mengunduh File Installer dari GitHub

Pertama, kita perlu mengunduh file masternya. Jangan khawatir, saya sudah siapkan link-nya agar teman-teman tidak tersesat.

  1. Buka browser favorit kalian dan langsung menuju ke halaman rilis Laragon di GitHub: https://github.com/leokhoa/laragon/releases
  2. Cari rilis untuk versi Laragon 6.0.0.
  3. Unduh file bernama laragon-wamp-6.0.0.exe. File ini adalah installer utama yang akan kita gunakan.
Gambar di atas merupakan dokumentasi langkah-langkah mengunduh file installer Laragon dari GitHub
Langkah 2: Menjalankan File Installer

Setelah file terunduh, saatnya kita jalankan file installer tersebut.

  1. Jalankan file laragon-wamp.exe yang sudah diunduh.
  2. Pilih bahasa instalasi (disarankan English), lalu klik OK.
  3. Nah, ini bagian penting! Saat diminta memilih lokasi instalasi, biarkan saja di C:\laragon. Kenapa? Ini untuk menghindari masalah permission yang sering terjadi di Windows dan akan membuat hidup kita lebih mudah nantinya.
  4. Pada jendela opsi berikutnya, teman-teman bisa biarkan semua centang default aktif. Opsi seperti "Add Notepad++" dan "Add to context menu" itu sangat membantu!
  5. Klik Next, lalu Install, dan biarkan prosesnya berjalan hingga selesai.

Install phpMyAdmin

Setelah laragon terinstall langkah selanjutnya yaitu menginstall phpMyAdmin melalui laragon

  1. Buka laragon yang sudah diinstall tadi
  2. Klik Menu lalu pilih Quick app lalu pilih phpMyAdmin
  3. Tunggu proses installasi hingga selesai
  4. Untuk mengakses phpMyAdmin bisa melalui menu di laragon atau bisa juga melalui browser dengan mengetikkan alamat http://localhost/phpmyadmin/
  5. Untuk login ke phpMyAdmin, gunakan username: root dan password dikosongkan saja (default)
  6. Selesai, sekarang kalian sudah bisa mengelola database dengan mudah menggunakan phpMyAdmin!
